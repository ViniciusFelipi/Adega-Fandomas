# 🍷 Adega Fandomas

Sistema de gerenciamento de uma adega desenvolvido em **Java (Swing - JFrame)** com integração a **Banco de Dados (JDBC)** e arquitetura baseada em **POO + DAO**.

---

## 📌 Sobre o projeto

O sistema permite o gerenciamento completo de bebidas de uma adega, possibilitando operações de cadastro, busca, atualização e exclusão.  
A interface foi desenvolvida em Java Swing (JFrame), com integração direta ao banco de dados.

---

## 🚀 Funcionalidades

- 🔎 Busca de bebidas por nome (filtro dinâmico)
- ➕ Cadastro de novas bebidas
- ✏️ Atualização de registros existentes
- ❌ Exclusão de bebidas
- 📋 Exibição em tabela (JTable)
- 💾 Integração com banco de dados (CRUD completo)
- 🧠 Estrutura baseada em POO e padrão DAO

---

## 🛠️ Tecnologias utilizadas

- Java
- Swing (JFrame)
- JDBC
- SQL (Banco de Dados)
- Programação Orientada a Objetos (POO)

---

## 🧱 Estrutura do projeto

- **InterfaceAdega** → Interface gráfica principal (JFrame)
- **Bebidas** → Classe modelo (entidade)
- **BebidasDAO** → Classe responsável pelas operações no banco de dados (CRUD)
